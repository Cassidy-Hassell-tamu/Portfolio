# Portfolio

A portfolio website demonstrating projects and design.

## Tech Stack

- React + Vite
- Tailwind CSS
- Modern web development practices

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

- `src/components/` - React components (Header, HeroSection, AboutSection, ProjectsSection, ContactSection, Footer)
- `src/App.jsx` - Main application component
- `src/main.jsx` - Application entry point
- `src/index.css` - Global styles with Tailwind directives

## Features

- Responsive, mobile-first design
- Semantic HTML structure
- Accessibility features (keyboard navigation, focus indicators, ARIA labels)
- Smooth scrolling navigation
- Contact form
- Project showcase with tech stack display
